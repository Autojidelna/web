# Začínáme

## Průvodce stahováním

1. Navštivte naši [stránku](/cs/download/) a stáhněte nejnovější verzi aplikace **Autojídelna**.
2. Po dokončení stahování otevřete soubor `autojidelna.apk`.
3. Pokračujte v instalaci.
